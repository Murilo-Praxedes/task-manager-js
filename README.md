# 📝 Task Manager (Gerenciador de Tarefas)

Aplicação web desenvolvida para gerenciamento de tarefas do dia a dia, permitindo adicionar, concluir, filtrar e remover tarefas de forma simples e eficiente.

---

## 🚀 Demonstração

👉 https://murilo-praxedes.github.io/task-manager-js/

---

## 📸 Preview

![Preview do projeto](preview.png)

---

## ✨ Funcionalidades

* ✅ Adicionar novas tarefas
* ✔️ Marcar tarefas como concluídas
* ❌ Remover tarefas
* 🔎 Filtrar por:

  * Todas
  * Concluídas
  * Pendentes
* 💾 Persistência de dados com LocalStorage
* 🎨 Interface moderna e responsiva

---

## 🛠️ Tecnologias utilizadas

* HTML5

* JavaScript (Vanilla JS)
* LocalStorage

## 📦 Como executar o projeto

1. Clone o repositório:

```bash
git clone https://github.com/Murilo-Praxedes/task-manager-js.git
```

2. Acesse a pasta:

```bash
cd task-manager-js
```

3. Abra o arquivo `index.html` no navegador
